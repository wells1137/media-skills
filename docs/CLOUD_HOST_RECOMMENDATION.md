# 云主机选购建议（跑 OpenClaw 网关）

跑 OpenClaw 网关 + Telegram 等：**1 核 1GB 内存** 足够，流量和 CPU 都很轻。下面按场景推荐。

---

## 推荐一：国内用户、要便宜稳定

| 厂商 | 产品 | 配置示例 | 价格 | 说明 |
|------|------|----------|------|------|
| **火山引擎** | 云服务器 ECS | 按量/包年，多规格可选 | 按量计费 | 字节跳动云，控制台清晰，国内多地域 |
| **腾讯云** | 轻量应用服务器 | 1 核 2G / 30G SSD / 30M 带宽 | ¥24–35/月 | 香港/新加坡节点，免备案，中文控制台，对国内延迟低 |
| **阿里云** | 轻量应用服务器（国际） | 2 核 0.5G 起 / 200M 峰值带宽 | ¥28/月起 | 香港等，免备案，和腾讯二选一即可 |

- 购买： [火山引擎 ECS](https://www.volcengine.com/product/ecs) / [腾讯云轻量](https://cloud.tencent.com/product/lighthouse) / [阿里云轻量](https://www.aliyun.com/product/swas)
- 选 **香港** 或 **新加坡** 机房，Telegram / 国外 API 连通性更好。
- 系统选 **Ubuntu 22.04 LTS**。

---

## 推荐二：要国际厂商、按量付费

| 厂商 | 配置 | 价格 | 说明 |
|------|------|------|------|
| **Vultr** | 1 核 1G / 25G SSD | $5–6/月 | 新加坡/东京/美西，按小时计费，随时删机 |
| **Linode (Akamai)** | 1 核 1G / 1TB 出站 | $5/月 | 新加坡等，流量便宜、稳定 |
| **DigitalOcean** | 1 核 1G / 25G SSD | $6/月 | 新加坡/班加罗尔，文档和生态好 |

- Vultr：https://www.vultr.com  
- Linode：https://www.linode.com  
- 同样选 **Ubuntu 22.04**，机房选 **新加坡** 或 **东京** 延迟一般更好。

---

## 推荐三：预算极低、先试水

- **CloudCone**、**Hostwinds** 等有 $2.5–3.5/月 档位，机房多在美西，国内延迟会高一些，适合先跑通再换。
- 若主要给国外用或自己挂梯子访问，可以接受。

---

## 配置建议（通用）

- **配置**：1 核 1GB 起即可；若同时跑别的服务，可上 1 核 2GB。
- **系统**：Ubuntu 22.04 LTS。
- **地域**：  
  - 人在国内、希望延迟低 → 腾讯云/阿里云 **香港**。  
  - 希望连 Telegram / 国外 API 稳 → **香港** 或 **新加坡**（国内外厂商都可选这两个）。
- **安全**：只开放 22（SSH）、必要时 80/443；密钥用环境变量，不要写进代码或提交 Git。

---

## 小结

- **国内 + 省心**：火山引擎 / 腾讯云 / 阿里云，按量或包月，选香港/新加坡对国外服务更友好。  
- **国际 + 灵活**：Vultr 或 Linode，新加坡/东京，$5–6/月。  
- 买好后在服务器上装 OpenClaw 网关、配好 `.env` 和 Telegram token，把原来本地的网关停掉即可。
